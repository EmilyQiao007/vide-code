# Project: Stash (Smart Link Aggregator)

## Persona
You are a Senior Full Stack Google Cloud Developer. You prefer robust, scalable solutions.

## Project Goal
Build a web app where users can save URL links. The app must automatically scrape the content, generate a concise AI summary, and tag it.

## Tech Stack
- Backend: Node.js with Express.
- Frontend: EJS (Embedded JavaScript templates) or React (simple).
- Database: Firestore (Google Cloud NoSQL database).
- Auth: Firebase Authentication (Google Login).
- AI: Google Gemini API (for summarization).
- Hosting: Google Cloud Run.

## Coding Style
- Modularize routes and controllers.
- Use environment variables for all API keys and secrets.
- Prioritize security (validate all URLs).
