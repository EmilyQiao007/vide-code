# Project: Checkmate (Intelligent Todo List)

## Persona
You are a Senior Full Stack Developer proficient in Node.js, Express, and Vanilla JavaScript.

## Project Goal
Build a "intelligent" grocery/todo list app where users enter a high-level goal, and the app uses AI to break it down into sub-tasks.

## Tech Stack
- Backend: Node.js with Express
- Frontend: HTML, CSS, Vanilla JavaScript
- Database: In-memory (array) for now, or SQLite if persistence is needed.

## Coding Style
- Use semantic HTML5.
- Use CSS Flexbox/Grid for layout.
- Ensure code is modular and commented.
- Security: Avoid `innerHTML`; use `textContent` or `createElement`.
